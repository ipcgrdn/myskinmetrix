# My Skin Metrix

## 초기 기획

### 문제 정의

**현재 피부 분석 시장의 문제점:**

1. **정확성 부족**: 대부분의 피부 분석은 단순 설문이나 일회성 측정에 의존해 정확도가 낮고, 환경·생활습관·스트레스 등 복합적 요인을 고려하지 못함
2. **접근성 한계**: 전문적인 피부 분석을 받으려면 고가의 피부과나 피부관리실 방문이 필요하며, 지속적인 관리가 어려움
3. **맞춤형 솔루션 부재**: 일반적인 피부 타입 분류(건성/지성/복합성/민감성)는 지나치게 단순화되어 개인별 특성을 반영하지 못함
4. **상업적 편향**: 기존 피부 분석 서비스는 특정 브랜드 제품 판매를 위한 도구로 활용되어 객관적 추천이 어려움
5. **지속적 관리 부족**: 한 번의 분석으로 끝나고 시간에 따른 피부 변화를 추적하는 솔루션이 부족함

**My Skin Metrix가 해결하고자 하는 핵심 가치:**

- 정확하고 종합적인 피부 분석으로 사용자가 자신의 피부를 깊이 이해할 수 있도록 지원
- 언제 어디서나 접근 가능한 전문적 피부 분석 서비스 제공
- 개인별 특성과 환경을 고려한 진정한 맞춤형 솔루션 제공
- 객관적이고 신뢰할 수 있는 제품 추천으로 불필요한 소비 방지
- 시간에 따른 피부 변화를 추적하여 지속적인 관리 지원

### 목표 고객

**주요 타겟 페르소나:**

1. **스킨케어 초보자 (20-25세)**
    - 특징: 대학생/사회초년생, 피부 관리의 중요성을 인식하기 시작
    - 니즈: 복잡하지 않은 기본적인 피부 케어 가이드, 저렴한 입문용 제품 추천
    - 동기: 자신에게 맞는 기초 제품 찾기, 피부 트러블 예방
2. **적극적 스킨케어 실천자 (25-35세)**
    - 특징: 경제력을 갖춘 직장인, 피부 관리에 시간과 비용 투자 의향 있음
    - 니즈: 과학적 분석 기반 맞춤형 솔루션, 효과적인 프리미엄 제품 추천
    - 동기: 나이 들어가는 피부 변화 대응, 최적의 제품 조합 찾기
3. **피부 문제 해결자 (15-40세)**
    - 특징: 여드름, 민감성, 건조함 등 특정 피부 고민이 있는 사용자
    - 니즈: 특정 피부 문제에 대한 정확한 진단과 맞춤 솔루션
    - 동기: 지속적인 피부 문제 해결, 효과적인 개선 방법 찾기
4. **뷰티 트렌드 얼리어답터 (18-28세)**
    - 특징: SNS 활동적, 최신 뷰티 트렌드 민감, 콘텐츠 공유 활발
    - 니즈: 재미있는 분석 경험, 공유하기 좋은 결과물, 새로운 제품 발견
    - 동기: 트렌드 선도, 피부 분석 경험 공유

**지역 및 시장 세분화:**

- 초기: 한국 시장 (스킨케어 관심도 높고 디지털 리터러시 우수)
- 확장: 아시아 시장 (일본, 중국, 대만, 태국 등 K-뷰티 인기 국가)
- 장기: 글로벌 시장 (다양한 인종의 피부 특성 데이터 확보 후)

**특별 타겟 그룹:**

- 남성 스킨케어 입문자 (기존 시장에서 소외된 블루오션)
- 환경/성분에 민감한 클린 뷰티 추구자
- 비용 효율적인 스킨케어 루틴을 찾는 가성비 추구자

**시장 검증 방법:**

- 페이스북/인스타그램 타겟 광고를 통한 랜딩 페이지 전환율 측정
- 각 페르소나별 소규모 인터뷰 (5-10명) 진행
- 베타 테스터 그룹 구성 및 초기 피드백 수집 (30-50명)

### 핵심 기능

**MVP (1단계) - 기본 분석 기능:**

- **설문 기반 피부 타입 검사**
    - 15개 이하의 핵심 질문으로 기본적인 피부 타입 판별
    - 생활 습관, 피부 고민, 환경 요인 등 다각적 접근
    - 결과: 5가지 주요 지표 점수 (유수분, 민감도, 색소침착, 노화도, 회복력)

**성장 단계 (2단계) - 시각적 분석 추가:**

- **이미지 기반 피부 타입 검사**
    - 셀카 이미지 분석으로 모공, 주름, 색소침착, 유·수분 측정
    - 다양한 조명 환경에서도 정확한 분석을 위한 보정 기술
    - 설문 결과와 이미지 분석 결과 통합 알고리즘

**확장 단계 (3단계) - 고급 기능 추가:**

- **심층 분석 (통합 모델)**
    - 시간에 따른 피부 변화 추적 대시보드
    - AI 기반 미래 피부 예측 시뮬레이션
    - 환경 요인 (날씨, 대기질, 습도)과 피부 상태 상관관계 분석
- **퍼스널 컬러 검사**
    - 얼굴 이미지 기반 퍼스널 컬러 진단
    - 피부톤에 맞는 메이크업/의류 색상 추천
    - 가상 메이크업 시뮬레이션

**차별화 요소:**

- 설문+이미지+환경 데이터의 통합 분석으로 정확도 향상
- 시간에 따른 피부 변화 추적 기능 (다른 앱들은 일회성 분석에 그침)
- 객관적인 제품 추천 알고리즘 (브랜드 편향 없음)
- 사용자 데이터 축적을 통한 지속적 모델 개선 및 정확도 향상
- 뭔가의 특별한 요소 ++ (게이미피케이션 요소?)

### 수익 모델

**기본 모델 (무료):**

- 간단한 설문 기반 기본 피부 타입 진단 (5가지 지표)
- 제한적인 일반 추천 정보 제공
- 기본 피부 관리 팁 제공
- *목적: 사용자 유입 및 데이터 수집, 유료 서비스로 전환 유도*

**프리미엄 기능 (유료):**

- **심층 분석 리포트 (9,900원)**
    - 상세한 피부 상태 분석 및 맞춤형 관리 가이드
    - 25가지 세부 지표 분석 결과
    - PDF/이미지 형식의 공유 가능한 상세 리포트
- **이미지 기반 분석 (19,900원)**
    - 얼굴 사진을 통한 피부 상태 정밀 분석
    - 설문+이미지 통합 분석 결과
    - 문제 영역 시각화 및 개선 시뮬레이션
- **정기 구독 플랜 (월 4,900원)**
    - 피부 상태 변화 추적 대시보드
    - 매주 업데이트되는 맞춤형 케어 가이드
    - 계절/환경 변화에 따른 피부 관리 조언

**제휴 마케팅:**

- 분석 결과에 기반한 맞춤형 제품 추천 (affiliate 수수료 10-15%)
- 추천 제품 구매 시 캐시백 또는 포인트 제공 (사용자 혜택)
- 정직한 추천 원칙: 평점 4점 이상, 실제 효과 검증된 제품만 추천

**B2B 수익 모델:**

- 화장품 기업에 익명화된 트렌드 리포트 제공 (월 구독형)
- 피부 분석 API 제공 (화장품 브랜드 자체 앱에 통합)
- 맞춤형 화장품 브랜드와의 협업 모델 (사용자 데이터 기반 제품 개발)

**수익화 전략:**

- 무료→유료 전환율 목표: 5-7% (업계 평균 3-4%)
- 초기 목표: 월 500명 유료 사용자 × 평균 15,000원 = 750만원/월
- 12개월 목표: 월 5,000명 유료 사용자 × 평균 12,000원 = 6,000만원/월

### 개발

**개발 방향:**

- **모바일 퍼스트** 설계 (반응형 웹앱, PWA 적용)
- **다국어 지원** 계획 (한국어 → 영어 → 중국어/일본어)

**기술 스택:**

- 프론트엔드: React/Next.js, TailwindCSS
- 백엔드: Node.js, Express, MongoDB
- AI/ML: TensorFlow.js, Python (서버사이드 분석)
- 인프라: Vercel, AWS Lambda

**개발 로드맵:**

- **1단계 (1-2개월)**: 기본 설문 기반 분석 시스템 개발
    - 핵심 설문 시스템 구축 (2주)
    - 기본 결과 알고리즘 개발 (2주)
    - 초기 UI/UX 디자인 및 구현 (2주)
    - 베타 테스트 및 수정 (2주)
- **2단계 (3-4개월)**: 이미지 분석 기능 추가 개발
    - 이미지 처리 모듈 개발 (4주)
    - AI 모델 학습 및 통합 (4주)
    - 설문+이미지 통합 분석 알고리즘 개발 (4주)
    - 테스트 및 개선 (4주)
- **3단계 (5-6개월)**: 통합 분석 모델 개발 및 고급 기능 구현
    - 변화 추적 시스템 개발 (4주)
    - 환경 데이터 통합 (4주)
    - 퍼스널 컬러 분석 추가 (4주)
    - 고급 추천 알고리즘 개발 (4주)
- **4단계 (7-12개월)**: 사용자 피드백 기반 개선 및 확장
    - 데이터 기반 지속적 모델 개선 (매월)
    - 사용자 인터페이스 최적화 (계속)
    - 추가 기능 개발 (사용자 요구에 따라)
    - 다국어 지원 추가 (8-10개월 차)

**사용자 피드백 및 개선 사이클:**

- **피드백 루프 설계**:
    - 앱 내 피드백 시스템 (별점, 피드백 폼)
    - 사용자 행동 분석 (사용 패턴, 이탈 지점 추적)
    - 정기적인 사용자 설문조사 (월 1회)
    - 핵심 사용자 인터뷰 (분기별)
- **데이터 기반 의사결정**:
    - 주간 핵심 지표 분석 (회원가입, 전환율, 체류시간, NPS)
    - 월간 기능 우선순위 조정
    - 분기별 제품 로드맵 검토
- **지속적 참여 전략**:
    - 정기적인 콘텐츠 업데이트 (주 1회 피부 관리 팁)
    - 시즌별 특별 이벤트 (봄/여름/가을/겨울 피부 관리)
    - 커뮤니티 참여 유도 (사용자 성공 사례 공유)
    - 알림 최적화 (개인화된 관리 알림, 과도하지 않은 빈도)

**MVP 출시 기준:**

- 설문 기반 기본 피부 타입 분석
- 간단한 이미지 업로드 및 기초 분석
- 기본 제품 추천 시스템
- 공유 가능한 결과 리포트
