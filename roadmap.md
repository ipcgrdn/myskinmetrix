## 기술 스택 선택

### 모노레포 아키텍처

- **모노레포 도구**: Turborepo
- **구조**: 앱(웹, API)과 공유 패키지(타입, UI, 설정 등)
- **빌드/배포**: 통합 CI/CD 파이프라인

### 프론트엔드

- **프레임워크**: Next.js (React 기반)
- **스타일링**: TailwindCSS + Shadcn UI
- **상태 관리**: React Context API → 필요시 Zustand
- **배포**: Vercel (무료 티어)

### 백엔드

- **프레임워크**: NestJS (TypeScript)
- **데이터베이스**: PostgreSQL
- **ORM**: Prisma (타입 안전성 및 마이그레이션 용이)
- **이미지 저장**: AWS S3 또는 자체 호스팅 스토리지
- **캐싱**: Redis (필요시)
- **AI 통합**: NestJS 내 child_process로 Python 스크립트 실행

### AI/ML

- **초기 이미지 분석**:
    - 클라이언트 측: TensorFlow.js + MediaPipe
    - 서버 측: Python 스크립트 (NestJS에 통합)
- **설문 분석**: 규칙 기반 알고리즘 (전문가 지식 기반)
- **고급 분석**: Hugging Face 모델 통합 (점진적 적용)

## 서버 인프라 구성

### 서버 아키텍처

- **단일 NestJS 서버**: 모든 백엔드 로직 및 API
- **Python 통합**: NestJS 내에서 child_process로 Python 스크립트 실행
- **Vercel**: Next.js 프론트엔드 호스팅 (자동 배포)

### NestJS와 Python 통합 예시

```tsx
// 서비스 클래스 (NestJS)
@Injectable()
export class PythonService {
  async analyzeSkin(imageUrl: string): Promise<any> {
    return new Promise((resolve, reject) => {
      const process = spawn('python3', [
        './packages/ml/analyze_skin.py',
        imageUrl
      ]);

      let result = '';
      process.stdout.on('data', (data) => {
        result += data.toString();
      });

      process.on('close', (code) => {
        if (code !== 0) {
          reject(new Error(`Python process exited with code ${code}`));
          return;
        }
        try {
          resolve(JSON.parse(result));
        } catch (e) {
          reject(new Error('Could not parse Python output'));
        }
      });
    });
  }
}

```

- **상세 분석 원할 때**: "더 자세한 분석을 보시겠습니까?"

### 비로그인 결제 흐름
